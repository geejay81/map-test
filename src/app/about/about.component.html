Technologies and Packages

Some of the technologies and packages used to create this site.

Angular

Angular 5 and Angular CLI were used to create the application.

Bulma

Bulma is a CSS framework used to tyle the pages.

Google Maps

Markers are used to represent Bike Docks in London.

HTML5

HTML5 is used to layout the pages, including semantic tags.

TfL

The latest data is drawn from the Transport for London API.
